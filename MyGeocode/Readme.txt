MyGeocode
 - 지오코딩 어플리케이션
 - 다른 액티비티가 띄워진 상태에서 뒤로가기를 눌렀을때 비정상 종료되는 예외처리 미구현


주소 단어로 검색하기

1. 주소로 검색하기에 체크하고 입력창 이동 버튼을 클릭한다.
2. 주소 입력칸에 검색할 단어를 입력한고, 검색 버튼을 누른다.
3. 메인 화면으로 넘어가면서 버튼 하단에 검색 결과가 출력된다.


위도, 경도값으로 검색하기

1. 좌표로 검색하기에 체크하고 입력창 이동 버튼을 클릭한다.
2. 위도와 경도 입력칸에 검색할 위도값과 경도값를 입력하고, 검색 버튼을 누른다.
3. 메인 화면으로 넘어가면서 버튼 하단에 검색 결과가 출력된다.
